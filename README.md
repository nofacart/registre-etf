# Registre — Portefeuille ETF

Page web personnelle pour suivre l'évolution de votre portefeuille ETF : bilans mensuels, trimestriels et annuels, répartition par ligne, plus/moins-value latente.

## Publier sur GitHub Pages

1. Créez un nouveau repo sur GitHub (public — requis pour GitHub Pages avec un compte gratuit).
2. Ajoutez le contenu de ce dossier à la racine du repo (le fichier `index.html` doit être à la racine, pas dans un sous-dossier).
3. Allez dans **Settings → Pages**.
4. Dans **Source**, choisissez la branche `main` (ou `master`) et le dossier `/ (root)`.
5. Cliquez **Save**. Votre page sera accessible après quelques minutes à :
   `https://votre-pseudo.github.io/nom-du-repo/`

## Vos données

- Les données sont stockées uniquement dans le navigateur (`localStorage`) de l'appareil utilisé — jamais dans le code, jamais sur GitHub.
- Pour retrouver vos données sur un autre appareil : utilisez les boutons **Exporter** / **Importer** en haut de la page, en passant par un dossier cloud synchronisé (Drive, iCloud, OneDrive...).
- Règle simple : **importer avant de saisir**, **exporter après avoir saisi**, pour toujours repartir de la version la plus à jour.
- Pensez à garder vos fichiers `.json` exportés quelque part en dehors du navigateur : c'est votre vraie sauvegarde.
